# miniProject-selectShop

Spring Boot 기반의 상품 선택 및 주문 미니 프로젝트입니다.


# 프로젝트 소개
- 상품 등록 및 조회 기능
- 상품 주문 기능
- JPA를 활용한 데이터 베이스 연동
- 주문 조회 및 전체 조회 페이징
- 전체 주문 시 N+1 문제 해결
- Stock(재고) 차감
- 동시 Stock(재고) 차감 시 문제 해결

# 기술 스택
- Java 17
- Spring Boot
- Spring Data JPA
- Gradle
- MySQL
